# Favicons

This folder is reserved for the web/browser favicon bundle.

## Target contents

- `favicon.svg`
- `favicon-16.png`
- `favicon-32.png`
- `favicon-48.png`
- `favicon-192.png`
- `favicon-512.png`

## Current state

The app currently points to `assets/images/favicon.png`.
Keep that file supported until the dedicated favicon bundle is exported here and referenced from `assets/manifest.json` / web config as needed.
