# Figma Export Handoff

Use this folder for lightweight, repo-safe Figma export metadata and batch notes.

## Keep here

- export manifests
- frame/component IDs
- release notes for exported batches
- links or identifiers that map shipped assets back to the editable design source

## Do not keep here

- private credentials
- huge binary exports that do not need version control
- duplicate production asset files that already live in `assets/logos/`, `assets/icons/`, `assets/images/`, `assets/lottie/`, or `assets/favicons/`

## Recommended file naming

- `brand-export-2026-04.json`
- `icons-export-2026-04.json`
- `illustrations-export-2026-04.json`
