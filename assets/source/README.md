# Source Assets

This folder is the canonical landing area for editable design-source materials that back shipped WeeJobs graphics.

## Purpose

Use `assets/source/` for repo-safe design handoff artifacts that describe or accompany exported visuals found elsewhere under `assets/`.

Typical contents:

- Figma export notes
- source-reference manifests
- lightweight SVG masters
- export checklists
- release notes for refreshed asset bundles

Do **not** commit large binary design files such as full PSD/AI archives without approval.

## Expected structure

- `assets/source/figma-exports/` — export bundles and source-reference manifests
- `assets/source/brand/` — logo-family notes and release records
- `assets/source/icons/` — icon-system source notes and naming inventory
- `assets/source/illustrations/` — illustration/export notes

## Rule of thumb

Every shipped asset family should eventually have a traceable source reference here or in an equivalent documented design system location.
