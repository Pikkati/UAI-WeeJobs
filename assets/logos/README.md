# Logos

This folder is reserved for the standardized WeeJobs logo family.

## Target contents

- `logo-primary.svg`
- `logo-stacked.svg`
- `logo-mark.svg`
- monochrome variants
- PNG fallbacks for shipped surfaces where needed

## Naming rules

- lowercase
- kebab-case
- prefix with `logo-`

## Current state

The app currently ships the PNG logo at `assets/images/weejobs-logo.png`.
That file should remain supported until the finalized vector logo family is exported here and registered in `assets/manifest.json`.
