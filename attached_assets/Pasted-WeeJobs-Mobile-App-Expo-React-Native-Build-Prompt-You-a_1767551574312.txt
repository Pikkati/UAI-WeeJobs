WeeJobs Mobile App – Expo (React Native) Build Prompt

You are building WeeJobs, a mobile-first handyman marketplace app for the Causeway Coast & Glens, Northern Ireland.

WeeJobs connects local customers with local tradespeople for small to medium jobs.
Tagline: “No Job Too Wee” — no task is too small.

This is an MVP focused on UX, flow, and structure.
Authentication will be mocked (no real auth yet).
Database: Supabase (tables + basic reads/writes only).

Core Concept

Customers post jobs quickly via the app

Tradespeople swipe through available jobs and accept ones they want

Messaging opens once a job is accepted

Admin has a lightweight dashboard (separate entry point)

Tech Stack

Expo (React Native)

TypeScript

Supabase (Postgres + client SDK)
